# Pathfinder

Pathfinder is a Python-based GUI application built with Tkinter that allows users to search for files and folders across their entire system.

## Features

- **System-wide Search**: Search for files and directories by name.
- **Real-time Results**: View results as they are found.
- **Quick Actions**:
    - Copy file/folder path to clipboard.
    - Open the file or folder location directly.
- **Clean Interface**: A simple and user-friendly GUI.

## Requirements

- Python 3.x
- Tkinter (usually comes with Python)
- `pyperclip` (for clipboard functionality)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/JedizLaPulga/pathfinder.git
   ```
2. Navigate to the project directory:
   ```bash
   cd pathfinder
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. Run the application:
   ```bash
   python main.py
   ```
2. Enter a search term in the search bar.
3. Press **Enter** or click the **Search** button.
4. Interact with the results using the side buttons or context menu (if implemented).
